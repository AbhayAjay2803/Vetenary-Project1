# Veterinary Health Assessment System

An AI-powered veterinary health assessment system that uses ensemble machine learning to predict animal health risks based on symptoms and patient information.

## 🚀 Features

- **Ensemble Machine Learning**: Combines 5 different models for robust predictions
- **Streamlit Web Interface**: User-friendly web application
- **AI-Generated Reports**: Professional veterinary reports using OpenAI GPT
- **Real-time Risk Assessment**: Immediate health risk evaluation
- **Model Performance Tracking**: Comprehensive model comparison and analytics

## 🏆 Best Performing Model in Colab

**Improved Structured Clinical Transformer (SCT)**
- Accuracy: 84.0%
- F1-Score: 84.3%
- AUC: 91.0%
- Precision: 82.8%
- Recall: 85.9%
